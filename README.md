# B140

RQ:

Is there a significant difference in the mean number of confirmed COVID-19 
cases between collective infection clusters (group = TRUE) and individual/isolated 
infection events (group = FALSE)?

Hypotheses:

H₀ (Null Hypothesis):
There is no difference in the mean number of confirmed cases between 
group = TRUE and group = FALSE infection events.

H₁ (Alternative Hypothesis):
There is a difference in the mean number of confirmed cases between 
group = TRUE and group = FALSE infection events.
